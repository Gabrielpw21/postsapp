--- PROJETO CRIDAO UTILIZANDO XAMPP COMO SERVIDOR LOCAL --- 

*Configuração Local:*

    Composer install
    Criação DB e configuração .env
    php artisan migrate / php artisan migrate:fresh --seed

*Buildar componentes:*

    npm i/npm install
    npm run watch/npm run dev/npm run prod

*Explicações do projeto:*

    Meu objetivo com esse projeto foi deixar o código mais breve e enxuto possível.

    *Em App/Services é onde está a maior parte do back-end que fiz deixando POO.*

    *No front-end utilizei o axios para fazer requests para a API no back-end com laravel.*

    *Na estilização utilizei o bootstrap e suas classes para deixar design mais responsivo sem ter que mexer no CSS diretamente.*
